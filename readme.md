### How to run

- Clone Repository
- `npm install --include=dev`

run with 

- `ts-node src/app.ts --help` or
- `npm run dev -- --help`

You can build it first too

- `npm run build`
- `npm run start -- --help`

### Dependecy / Library Use

- Winston for logger
- mocha & chai for test

### Testing

Test All Module

```npm run test:all``` 

Test Specific file

```npm run test -- ./src/test/fizzbuzz.test.ts```

### TS Coverage [ 100% ]

`npm run ts-coverage`

to show report
